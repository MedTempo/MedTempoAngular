# MedTempoAngular
### Comandos para a instalação

- Instale os módulos npm 

~~~comand
npm i
~~~


- Abra os códigos no seu editor preferido
  
~~~comand
code .
~~~
ou
~~~comand
codium .
~~~
ou qualquer outro editor


- Rode com Ionic
  
~~~comand
ionic serve --lab
~~~
ou com o script
  
~~~comand
npm run dev
~~~



### Comando para o build Android

- Lembre-se de instalar as dependências do Java
